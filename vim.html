Input

insert mode - Ctrl-r reg = insert contents of register reg

normal mode

  • "ap = insert contents of register a at cusor

visual mode

  • p = change selection with contents of default register
  • "ap" =

Search

normal mode

  • * = search current word

visual mode

  • = search the selected string(to find a plugin)

Conception

verbs

  • v = visual
  • c = change
  • d = delete
  • y = yank/copy

modifier

  • i = inside
  • a = around
  • t = till..find a character(not include the char)
  • f = find..like till except including tha char(include the char)
  • / = search..find a string/regx(encluded)

text object

  • w = word
  • s = sentence
  • p = paragraph
  • b = block/parenthese
  • t = tag, work for html/xml

operate

  • diw = delete inside word(delete the current word)
  • cis = change inside sentence(change the current sentence)
  • ci" = change inside quote(change a string inside quotes)
  • c/foo = change search foo(change untile next occurrence of 'foo')
  • ctX = chage till X(change evering till X)
  • vap = visual arround paragraph(visually select this paragraph)

register

  • "a = use register a in the following operation, a can be another char
  • 0 = yank register
  • " = the default register
  • % = the current filename
  • * = the cliboard contents(X11:primary selection)
  • + = the cliboard contents
  • / = the last pattern
  • : = the last command-line
  • . = the last inserted text
  • - = the last small(less than a line) delete
  • help i_CTRL-R = help file

Others

  • :noh = reset hightlight when you search
